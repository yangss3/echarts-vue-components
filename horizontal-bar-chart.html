# HorizontalBarChart

横向的柱状图，这个组件主要用于同一系列中各类目的数据对比，或两个不同系列之间的数据对比。

# API

关于 series

数据类型如下：

series: {
  data: number[], // 系列的数据
  name?: string, // 系列名
  color?: string // 柱条的颜色，不提供会默认使用调色盘的颜色
}[]

当type 为 "two-way" 时，只支持两个系列的对比，如果 series 提供超过两个系列的数据，也只有前两个系列会生效：

[
  {
    data: [23, 24, 34, 56],
    name: '美国',
    color: 'cyan',
  },
  {
    name: '中国',
    data: [34, 13, 45, 32],
  },
  // 当 type='two-way'时，后面的系列不会生效
  {
    data: [13, 44, 34, 26],
    name: '日本',
  },
];

labelLeft, gridWidth, gridTop

根据图表容器尺寸的不同，你需要调整这三个属性来调整元素的位置
